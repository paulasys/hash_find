# hash_find
O Hash Find é um programa que gera, a partir de uma wordlist comum, uma wordlist com as palavras junto ao seu respectivo hash (com o tipo definido pelo usuário). E depois compara o hash informada para tentar encontrar correspondência na lista gerada.
```
$./hash_find <wordlist>
Informe o hash
<hash>

Informe o tipo de hash (sha1sum, md5sum, etc)
<sha1sum, sha256sum, sha512sum, md5sum>

```
